## API

- 登录

```json
method: post
url:   /login
# 请求参数
# params（可选）
"params":
{
  "username":"admin",
  "password":"123456"
}
# body（可选）
"form-data":
{
  "username":"admin",
  "password":"123456"  
 }
```

- 获取用户信息

```json
method: get
url:   /info
# 请求头
headers:{
    "token":"xxxxx"
}
```

- 退出登录

```json
method: get
url:   /logout
# 请求头
headers:{
    "token":"xxxxx"
}
```
